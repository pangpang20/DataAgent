# 示例

## 示例 1（MySQL）
任务：从 orders 表按月统计销售额
SQL: SELECT DATE_FORMAT(order_date, '%Y-%m') AS month, SUM(total_amount) AS sales FROM orders WHERE order_date >= '2025-01-01' AND order_date <= '2025-12-31' GROUP BY DATE_FORMAT(order_date, '%Y-%m') ORDER BY month

## 示例 2（Dameng）
任务：从 orders 表按月统计销售额
SQL: SELECT TO_CHAR(order_date, 'YYYY-MM') AS month, SUM(total_amount) AS sales FROM orders WHERE order_date >= TO_DATE('2025-01-01', 'YYYY-MM-DD') AND order_date <= TO_DATE('2025-12-31', 'YYYY-MM-DD') GROUP BY TO_CHAR(order_date, 'YYYY-MM') ORDER BY month

## 示例 3（MySQL）
任务：按省份统计用户数
SQL: SELECT province, COUNT(user_id) AS user_count FROM users GROUP BY province ORDER BY user_count DESC

## 示例 4（Dameng）
任务：找出销售额最高的 5 个产品
SQL: SELECT product_id, SUM(amount) AS total_sales FROM order_items GROUP BY product_id ORDER BY total_sales DESC LIMIT 5
