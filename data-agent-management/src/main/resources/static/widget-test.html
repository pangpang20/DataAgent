DataAgent Widget 测试页面

欢迎使用 DataAgent AI 助手

这是一个测试页面，用于演示 DataAgent Widget 的嵌入功能。

您应该能在页面右下角看到一个浮动的聊天按钮，点击即可与 AI 助手对话。

注意：
  • 请确保已经构建了 widget.js 文件（运行 npm run build:widget）
  • 请将下面配置中的 agentId 和 apiKey 替换为您的实际值
  • 确保后端服务正在运行（默认 http://localhost:8065）
